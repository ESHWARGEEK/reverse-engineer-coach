## What is Reverse Engineer Coach?

The Reverse Engineer Coach is an AI-powered learning platform that helps developers understand complex codebases by:

- **Intelligent Repository Discovery**: Find the best open-source repositories to learn from based on your learning goals
- **Personalized Learning Paths**: Generate custom curricula tailored to your skill level and interests
- **Interactive Code Analysis**: Get AI-powered explanations and guidance as you explore code
- **Progress Tracking**: Monitor your learning journey with detailed project management
