# Getting Started with Reverse Engineer Coach

This guide will walk you through creating your account and starting your first learning project.

## Prerequisites

Before you begin, you'll need:

1. **GitHub Account**: A GitHub account for accessing repositories
2. **GitHub Personal Access Token**: For repository analysis (we'll help you create this)
3. **AI Service API Key**: OpenAI API key or similar for AI-powered coaching
4. **Modern Web Browser**: Chrome, Firefox, Safari, or Edge

## Step 1: Create Your Account

1. **Visit the Registration Page**
   - Navigate to the Reverse Engineer Coach platform
   - Click "Sign Up" or "Register"

2. **Fill in Your Information**
   - **Email**: Use a valid email address (you'll need to verify it)
   - **Password**: Create a strong password (minimum 8 characters, include numbers and symbols)
   - **Confirm Password**: Re-enter your password

3. **Add Your API Credentials**
   - **GitHub Token**: Your personal access token (see [API Credentials Setup](api-credentials.md))
   - **AI API Key**: Your OpenAI or compatible AI service key

4. **Complete Registration**
   - Review the terms of service
   - Click "Create Account"
   - Check your email for verification (if required)

## Step 2: Verify Your Setup

After registration, the system will:

1. **Test Your GitHub Token**: Verify it can access public repositories
2. **Test Your AI API Key**: Confirm it can make API calls
3. **Create Your Profile**: Set up your personal learning dashboard

If any credentials fail validation, you'll see clear error messages with instructions to fix them.

## Step 3: Start Your First Learning Project

1. **Access Your Dashboard**
   - After successful registration, you'll be redirected to your personal dashboard
   - This is your home base for managing all learning projects

2. **Define Your Learning Goal**
   - Click "Start New Project" or "Create Learning Project"
   - Enter what you want to learn in natural language
   - Examples:
     - "React hooks and state management"
     - "Microservices architecture patterns"
     - "GraphQL API design"
     - "Docker containerization"

3. **Choose a Repository**
   - The system will search GitHub for the best learning repositories
   - Review the suggested repositories with quality scores
   - Select the one that best matches your learning goals

4. **Begin Learning**
   - The AI coach will analyze the repository
   - A personalized curriculum will be generated
   - Start exploring the code with guided assistance

## Step 4: Navigate Your Learning Environment

### Dashboard Overview
- **Active Projects**: See all your current learning projects
- **Progress Tracking**: Monitor completion status and time spent
- **Recent Activity**: Quick access to recently viewed projects
- **Learning Statistics**: Track your overall progress and achievements

### Workspace Features
- **Code Explorer**: Browse repository files with AI explanations
- **Task List**: Follow structured learning tasks
- **AI Coach**: Get real-time help and explanations
- **Notes**: Save important insights and learnings

### Profile Management
- **Account Settings**: Update email, password, and preferences
- **API Credentials**: Manage and update your tokens securely
- **Learning History**: Review completed projects and achievements
