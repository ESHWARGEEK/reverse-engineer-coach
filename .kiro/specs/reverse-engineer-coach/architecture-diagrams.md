### AWS Service Mapping

**Frontend Layer:**
- **Route 53**: DNS management and traffic routing
- **CloudFront**: Global CDN for React application delivery
- **WAF**: Web application firewall for security
- **S3**: Static website hosting for React frontend

**Application Layer:**
- **Application Load Balancer (ALB)**: Traffic distribution
- **ECS Fargate**: Containerized FastAPI backend services
- **API Gateway**: REST API management and throttling
- **Lambda**: Serverless functions for lightweight operations
- **SageMaker**: AI/ML model training and inference
- **Bedrock**: Managed LLM services for AI coaching

**Data Layer:**
- **RDS PostgreSQL**: Primary application database
- **ElastiCache Redis**: Session and application caching
- **S3**: Object storage for code repositories and assets
- **OpenSearch**: Analytics and search capabilities
- **EventBridge**: Event-driven architecture coordination

**Security & Operations:**
- **IAM**: Identity and access management
- **Secrets Manager**: Secure credential storage
- **CloudWatch**: Monitoring and logging
- **X-Ray**: Distributed tracing and performance monitoring

### Detailed Use Case Specifications

**UC-001: Create Learning Project**
- **Actor**: Junior/Senior Developer
- **Description**: User selects architecture topic and target repository
- **Preconditions**: User authenticated, valid GitHub access
- **Main Flow**: 
  1. Select career track (Senior/Staff/Principal)
  2. Choose architecture specialization
  3. Input target repository URL
  4. System validates repository access
  5. Project created with initial assessment

**UC-002: Analyze Target Repository**
- **Actor**: System (AI Engine)
- **Description**: Intelligent analysis of production codebase
- **Main Flow**:
  1. Clone/access repository via MCP client
  2. Extract architectural patterns
  3. Identify core structural elements
  4. Generate complexity assessment
  5. Create learning path recommendations

**UC-003: Generate Curriculum**
- **Actor**: System (AI Curriculum Engine)
- **Description**: Create competency-based learning path
- **Main Flow**:
  1. Analyze extracted patterns
  2. Map to industry competency frameworks
  3. Generate progressive task sequence
  4. Create portfolio deliverables
  5. Set assessment criteria
